# Agent: 재무 분석가

> 객관적 재무 데이터 수집 (해석 없이 팩트만)

## 역할

`frameworks/` 디렉토리에 정의된 투자자 프레임워크의 지표들을 수집하고 정리한다.
**주관적 해석이나 투자 판단은 하지 않는다.**

## 참조 프레임워크

- `frameworks/peter_lynch.md` → PEG, 성장률, 부채비율
- `frameworks/warren_buffett.md` → ROE, Owner Earnings, 이익 일관성
- `frameworks/philip_fisher.md` → 이익률 추세, R&D 효율
- `frameworks/charlie_munger.md` → ROIC, 단위 경제학

---

## 수집 항목

### 1. Lynch 지표

| 지표 | 수집 내용 |
|------|----------|
| PEG Ratio | P/E ÷ EPS 성장률 |
| PEGY Ratio | P/E ÷ (EPS 성장률 + 배당수익률) |
| EPS 성장률 | YoY, 3년/5년 CAGR |
| 부채비율 | 총부채 ÷ 자기자본 |
| 기관 보유율 | 기관투자자 지분율 |

### 2. Buffett 지표

| 지표 | 수집 내용 |
|------|----------|
| ROE | 순이익 ÷ 자기자본 (5년 추이) |
| ROE 일관성 | 5년 표준편차 |
| Owner Earnings | 순이익 + 감가상각 - CapEx - 운전자본변화 |
| 이익 연속성 | 연속 흑자 연수 |
| 부채/순이익 | 장기부채 ÷ 연간 순이익 |

### 3. Fisher 지표

| 지표 | 수집 내용 |
|------|----------|
| 영업이익률 | 영업이익 ÷ 매출 (5년 추이) |
| 영업이익률 변화 | 개선/악화 방향 |
| R&D/매출 | 연구개발비 비율 |
| 매출총이익률 | 매출총이익 ÷ 매출 |

### 4. Munger 지표

| 지표 | 수집 내용 |
|------|----------|
| ROIC | NOPAT ÷ 투하자본 |
| FCF 전환율 | FCF ÷ 순이익 |
| 주식 희석률 | 발행주식수 변화 (5년) |

### 5. 공통 지표

| 지표 | 수집 내용 |
|------|----------|
| 시가총액 | 현재 시총 |
| EV | 시총 + 순부채 |
| P/E | 주가 ÷ EPS |
| P/S | 주가 ÷ 주당매출 |
| P/B | 주가 ÷ BPS |
| FCF | 영업현금흐름 - CapEx |
| FCF Yield | FCF ÷ 시총 |
