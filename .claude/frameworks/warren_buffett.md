# 워렌 버핏 (Warren Buffett) 프레임워크

> 출처: Berkshire Hathaway 주주서한, "The Essays of Warren Buffett"
> 핵심 철학: 훌륭한 기업을 적정 가격에

## 투자 철학

- **능력의 범위(Circle of Competence)**: 이해할 수 있는 사업에만 투자
- **해자(Moat)**: 지속 가능한 경쟁 우위를 가진 기업
- **장기 보유**: "영원히 보유할 주식을 사라"
- **안전마진**: 내재가치 대비 할인된 가격에 매수

## 4가지 투자 기준

버핏이 인수/투자 시 사용하는 기준:

1. **이해 가능한 사업** (Understandable)
2. **유리한 장기 전망** (Favorable Long-term Prospects)
3. **정직하고 유능한 경영진** (Honest and Competent Management)
4. **매력적인 가격** (Attractive Price)

## 핵심 재무 지표

### Owner Earnings (소유주 이익)

버핏이 1986년 주주서한에서 제시한 개념:

```
Owner Earnings = 순이익
                + 감가상각/상각
                - 유지 CapEx (Maintenance CapEx)
                - 운전자본 증가분
```

순이익보다 실제 주주에게 귀속되는 현금을 더 정확히 반영

### ROE (자기자본이익률)

| 수준 | 해석 |
|------|------|
| ≥ 20% | 탁월 |
| 15-20% | 우수 |
| 10-15% | 양호 |
| < 10% | 부진 |

**중요**: 부채 없이 높은 ROE를 달성하는 기업 선호

### ROE 일관성

| 조건 | 기준 |
|------|------|
| 5년 연속 | ROE ≥ 15% |
| 10년 평균 | ROE ≥ 15% |
| 표준편차 | 낮을수록 양호 |

### 이익의 예측 가능성

| 지표 | 기준 |
|------|------|
| 10년 연속 흑자 | 필수 |
| 이익 성장 | 꾸준한 우상향 |
| 이익 변동성 | 낮을수록 양호 |

### 부채 수준

| 지표 | 기준 |
|------|------|
| 부채비율 | < 50% |
| 장기부채/순이익 | < 5년치 |
| 이자보상배율 | > 10x |

버핏은 **무차입 또는 저부채** 기업을 강하게 선호

### 이익률

| 지표 | 기준 |
|------|------|
| 영업이익률 | 동종업계 상위 |
| 순이익률 | 지속적 개선 |

높은 이익률 = 경쟁 우위의 증거

## 체크리스트

### 사업 품질

- [ ] 사업을 10분 안에 이해할 수 있는가
- [ ] 10년 후에도 존재할 사업인가
- [ ] 지속 가능한 경쟁 우위(해자)가 있는가
- [ ] 가격 결정력이 있는가
- [ ] 자본 재투자 없이 이익 성장이 가능한가

### 경영진

- [ ] 경영진이 정직하고 유능한가
- [ ] 자본 배분을 합리적으로 하는가
- [ ] 주주 이익을 우선시하는가
- [ ] 자사주를 보유하고 있는가

### 재무 상태

- [ ] ROE ≥ 15% (5년 지속)
- [ ] 부채 수준이 낮은가
- [ ] FCF가 꾸준히 양수인가
- [ ] Owner Earnings가 성장하는가
- [ ] 유지 CapEx가 낮은가 (자본 경량 사업)

### 밸류에이션

- [ ] 내재가치 대비 할인된 가격인가
- [ ] 장기 보유 시 만족스러운 수익률이 예상되는가

## 버핏 명언

> "좋은 기업을 적정 가격에 사는 것이, 적정 기업을 좋은 가격에 사는 것보다 훨씬 낫다."

> "해자가 없는 기업은 성이 없는 것과 같다."

> "다른 사람들이 탐욕스러울 때 두려워하고, 다른 사람들이 두려워할 때 탐욕스러워라."

> "리스크는 자신이 무엇을 하고 있는지 모르는 데서 온다."
