# 피터 린치 (Peter Lynch) 프레임워크

> 출처: "One Up on Wall Street", "Beating the Street"
> 핵심 철학: GARP (Growth at a Reasonable Price)

## 투자 철학

- **자신이 아는 것에 투자하라**: 일상에서 발견한 기업에 주목
- **성장주를 합리적 가격에**: 성장성과 밸류에이션의 균형
- **10배 수익(Ten-bagger)**: 장기 보유로 대박 종목 발굴

## 기업 분류 체계

| 유형 | EPS 성장률 | 특징 |
|------|-----------|------|
| **Fast Growers** | 20-50% | 소형 공격적 성장주, 주력 투자 대상 |
| **Stalwarts** | 10-20% | 대형 우량주, 안정적 성장 |
| **Slow Growers** | < 10% | 성숙기 기업, 배당 중심 |
| **Cyclicals** | 변동적 | 경기 민감주, 타이밍 중요 |
| **Turnarounds** | 회복 중 | 위기 극복 기업 |
| **Asset Plays** | - | 숨겨진 자산 보유 기업 |

## 핵심 재무 지표

### PEG Ratio (Price/Earnings to Growth)

```
PEG = P/E Ratio ÷ EPS 성장률(%)
```

| PEG 값 | 해석 |
|--------|------|
| < 0.5 | 매우 저평가 |
| 0.5 - 1.0 | 저평가 (매수 고려) |
| 1.0 | 적정 가치 |
| > 1.0 | 고평가 |
| > 2.0 | 매우 고평가 |

### PEGY Ratio (배당 포함)

```
PEGY = P/E Ratio ÷ (EPS 성장률 + 배당수익률)
```

배당주 평가 시 사용

### 부채비율 (Debt to Equity)

| 수준 | 해석 |
|------|------|
| < 30% | 매우 양호 |
| 30-50% | 양호 |
| 50-80% | 주의 |
| > 80% | 위험 |

### 기타 지표

| 지표 | 기준 |
|------|------|
| ROE | ≥ 15% |
| 재고/매출 비율 | 감소 추세 양호 |
| 현금 포지션 | 순현금 선호 |
| 기관 보유율 | 낮을수록 기회 (아직 덜 알려진 종목) |

## 체크리스트

### 정량적 기준

- [ ] PEG < 1.0
- [ ] 부채비율 < 80% (이상적: < 50%)
- [ ] ROE ≥ 15%
- [ ] EPS 성장률 양수 (Fast Grower: 20%+)
- [ ] 재고 증가율 < 매출 증가율
- [ ] FCF 양수

### 정성적 기준

- [ ] 사업을 한 문장으로 설명 가능
- [ ] 경쟁 우위가 명확함
- [ ] 경영진이 자사주 매입 중
- [ ] 기관 보유율이 과도하지 않음
- [ ] 시장의 관심 밖에 있음 (아직 스토리가 안 퍼짐)

## 위험 신호 (Red Flags)

- 핫섹터의 핫스톡 (과열)
- 과도한 다각화 (diworsification)
- 위스퍼 스톡 (소문만 무성)
- 공급업체 의존도가 높은 기업
- 고객 집중도가 높은 기업

## 린치 명언

> "주식 뒤에는 기업이 있다. 기업이 잘되면 주식도 잘된다."

> "아마추어도 전문가를 이길 수 있다. 단, 자신이 아는 것에 투자한다면."

> "내가 가장 좋아하는 종목은 월가가 무시하는 단순하고 지루한 사업이다."
