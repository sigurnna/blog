# 찰리 멍거 (Charlie Munger) 프레임워크

> 출처: "Poor Charlie's Almanack", Berkshire/Daily Journal 연례 미팅
> 핵심 철학: 멀티 디시플린 사고 + 역발상

## 투자 철학

- **Latticework of Mental Models**: 다양한 학문의 모델을 격자처럼 엮어 판단
- **역발상 (Inversion)**: "어떻게 성공할까"보다 "어떻게 망할까"를 먼저 생각
- **능력의 범위**: 모르는 것은 인정하고 피한다
- **인내**: "큰 돈은 사고파는 게 아니라 기다림에서 나온다"

## 4가지 필터

멍거의 투자 의사결정 4단계:

### 1. 이해 가능한가? (Can I understand it?)

- 사업 모델을 명확히 설명할 수 있는가
- 5년 후 이 기업이 어떻게 돈을 벌지 예측 가능한가
- **통과 못하면**: 즉시 패스

### 2. 지속 가능한 경쟁 우위가 있는가? (Durable competitive advantage?)

- 해자의 종류와 강도
- 해자가 넓어지고 있는가, 좁아지고 있는가
- **통과 못하면**: 패스

### 3. 유능하고 정직한 경영진인가? (Able and trustworthy management?)

- 자본 배분 능력
- 주주 이익 vs 경영진 이익 정렬
- 과거 약속 이행 여부
- **통과 못하면**: 패스

### 4. 가격이 합리적인가? (Sensible price?)

- 내재가치 대비 충분한 안전마진
- 장기 보유 시 만족스러운 수익률
- **통과 못하면**: 기다림

## 핵심 재무 지표

### ROIC (투하자본이익률)

```
ROIC = NOPAT ÷ Invested Capital
     = 세후영업이익 ÷ (순운전자본 + 순고정자산)
```

| 수준 | 해석 |
|------|------|
| > 20% | 탁월한 자본 효율 |
| 15-20% | 우수 |
| 10-15% | 양호 |
| < 10% | 자본 파괴 가능성 |

**중요**: ROIC > WACC 지속 여부

### 단위 경제학 (Unit Economics)

| 지표 | 의미 |
|------|------|
| 고객당 수익 | 개별 고객에서 얻는 가치 |
| 고객 획득 비용 (CAC) | 고객 1명 확보 비용 |
| LTV/CAC | 고객 생애가치 / 획득비용 (> 3x 양호) |
| 한계 이익률 | 추가 판매당 이익률 |

### 현금 전환

| 지표 | 기준 |
|------|------|
| FCF 전환율 | FCF / 순이익 > 100% |
| 현금 전환 주기 | 짧을수록 양호 |

## 해자 (Moat) 분석

멍거/버핏이 인정하는 해자 유형:

| 해자 유형 | 설명 | 예시 |
|-----------|------|------|
| **브랜드** | 가격 프리미엄 가능 | 코카콜라, 애플 |
| **전환 비용** | 이탈이 어려움 | 엔터프라이즈 소프트웨어 |
| **네트워크 효과** | 사용자 많을수록 가치 증가 | 비자, 메타 |
| **비용 우위** | 경쟁사 대비 낮은 원가 | 코스트코, GEICO |
| **규모의 경제** | 규모가 클수록 효율적 | 아마존 |
| **규제 장벽** | 진입 허가 필요 | 철도, 유틸리티 |

### 해자 점검 질문

- 해자가 넓어지고 있는가, 좁아지고 있는가?
- 해자를 파괴할 수 있는 기술/경쟁자가 있는가?
- 해자 유지에 얼마나 재투자가 필요한가?

## 위험 신호 (Red Flags)

### 절대 투자하지 않는 경우

- [ ] 사업 모델이 불명확
- [ ] 만성적인 주식 희석
- [ ] 공격적 회계 처리
- [ ] 고객 집중도 과다
- [ ] 성장으로 포장된 부실한 단위 경제학
- [ ] 해자 없는 commodity 사업
- [ ] 홍보성 경영진
- [ ] 낙관적 시나리오에 의존하는 레버리지

### 패스해야 할 때

- 논리를 간단히 설명할 수 없으면
- 해자의 지속성이 하나의 가정에 의존하면
- 밸류에이션에 실수 여지가 없으면
- 이미 보유한 종목보다 낫지 않으면

## 체크리스트

### 사업 분석

- [ ] 한 문단으로 투자 논리 설명 가능
- [ ] 해자가 명확하고 지속 가능
- [ ] 단위 경제학이 건전
- [ ] ROIC > 15% (지속적)

### 경영진 분석

- [ ] 자본 배분 이력이 우수
- [ ] 경영진 인센티브가 주주와 정렬
- [ ] 과거 가이던스 달성률
- [ ] 어려울 때도 투명하게 소통

### 밸류에이션

- [ ] 보수적 가정으로도 매력적
- [ ] 실수해도 큰 손실 없는 가격
- [ ] 최선의 대안보다 나은가

### 리스크

- [ ] 주요 리스크 3가지 나열 가능
- [ ] 최악의 시나리오에서도 감당 가능
- [ ] 기본 가정(base rate)이 현실적

## 멍거 명언

> "역발상하라, 항상 역발상하라. (Invert, always invert.)"

> "낚싯대를 물고기가 있는 곳에 던져라."

> "단순함을 추구하되, 단순함을 불신하라."

> "좋은 사업을 적정 가격에 사는 것이, 적정 사업을 좋은 가격에 사는 것보다 낫다."

> "모든 현명한 투자는 가치 투자다. 지불한 것보다 더 많이 받는 것."

> "큰 돈은 사고파는 데서 나오지 않는다. 기다림에서 나온다."

---

**멍거의 조언**: "각 기업마다 다른 체크리스트와 멘탈 모델이 필요하다. '이 세 가지만 보면 된다'고 쉽게 말할 수 없다. 스스로 체득해야 평생 써먹는다."
