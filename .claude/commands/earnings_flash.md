# /earnings-flash

> 실적 발표 직후 빠른 분석 및 액션 판단

## 목표

실적 발표 직후 **30분 내** 다음을 파악:
1. 숫자가 좋았나/나빴나 (정량)
2. 내 투자 thesis가 여전히 유효한가 (정성)
3. 지금 뭘 해야 하나 (액션)

## 입력

```
/earnings-flash [티커] [분기] (예: /earnings-flash IREN FY25Q2)
```

**필수 참조:**
- `investor_profile.md` - 투자자 성향
- `portfolios.md` - 보유 현황
- `reports/[티커]/` - 기존 분석 (있는 경우)

---

## 아키텍처

```
[1단계: 실적 수치 수집]
    ├─ 헤드라인 (매출, EPS, 가이던스)
    ├─ 컨센서스 대비 Beat/Miss
    └─ YoY/QoQ 비교
              │
[2단계: 컨텍스트 분석]
    ├─ 경영진 코멘트 핵심
    ├─ 새로운 정보/변화
    └─ 시장 반응 (After-hours)
              │
[3단계: Thesis 점검]
    ├─ 기존 투자 논리 유효성
    ├─ 강화된 점 / 약화된 점
    └─ 리스크 변화
              │
[4단계: 액션 판단]
    └─ 추가 매수 / 홀딩 / 비중 축소 / 재검토
```

## 수집 항목

### 1단계: 실적 수치

| 항목 | 수집 내용 |
|------|----------|
| 매출 | 실적, 컨센서스, 서프라이즈 % |
| EPS | 실적, 컨센서스, 서프라이즈 % |
| 핵심 KPI | 산업별 핵심 지표 (해시레이트, MAU, GMV 등) |

|------|----------|

| 시장 반응 | After-hours 주가, 거래량 |

| 항목 | 수집 내용 |

## 주의사항

### 해야 할 것
- 실적 발표 자료 (8-K, Press Release) 직접 확인
- Earnings call transcript에서 경영진 발언 인용
- 기존 분석 (`reports/[티커]/`)과 연결하여 thesis 점검
- 투자자 프로필 (10배 성장주, 공격적) 반영한 액션 제안

### 하지 말 것
- 숫자만 나열하고 끝내기 (반드시 "그래서 뭐?"에 답할 것)
- 시장 반응에 휩쓸린 판단 (과민 반응 여부 냉정히 분석)
- 애매한 결론 ("좀 더 지켜봐야..." 금지, 명확한 액션 제시)

### 데이터 소스
- 실적 발표: SEC EDGAR (8-K), 기업 IR 페이지
- 컨센서스: Yahoo Finance, Seeking Alpha, Bloomberg
- Earnings call: Seeking Alpha Transcripts, The Motley Fool
- After-hours: Yahoo Finance, Google Finance

## 확장 사용

### Persona 적용 (선택)

기본은 중립 분석이지만, 특정 관점이 필요하면:

```
"이번 IREN 실적 Burry 관점으로 리스크 체크해줘"
→ 역발상 관점에서 시장이 놓친 리스크 집중 분석

"세상학개론 관점으로 내러티브 변화 봐줘"
→ AI 인프라 내러티브가 강화됐는지 점검
```

### 연속 분석

```
"지난 3개 분기 실적 트렌드 정리해줘"
→ 분기별 핵심 지표 추이 + thesis 변화 추적
```
