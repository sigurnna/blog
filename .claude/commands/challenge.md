# /challenge

> 7개 페르소나의 날카로운 반론으로 Thesis 스트레스 테스트

## 목표

**확증 편향 방지**를 위해:
1. 7개 페르소나가 **날카롭게** thesis 공격
2. 내가 **못 본 리스크** 발견
3. 논리의 **약점** 노출 → 보강 기회

## 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **날카롭게** | 예의 없이 약점을 파고듦 |
| **각자 관점으로** | 7개 페르소나가 자신의 프레임으로 공격 |
| **구체적으로** | "좀 위험해 보인다" 금지, 정확히 뭐가 문제인지 |
| **건설적으로** | 공격 후 "이걸 해결하려면" 힌트 제시 |

---

## 입력

```
/challenge [티커]
```

**전제 조건:** `research/[TICKER]/thesis_*.md` 파일이 존재해야 함

없으면: "먼저 /coach [티커]로 thesis를 정리해주세요."

**예시:**
```
/challenge IREN
/challenge PLTR
```

```
    ▼

    ├─ Howard Marks: 사이클/타이밍
    ├─ Druckenmiller: 매크로 리스크
    └─ Bill Ackman: 촉매 부재/가치 함정
    ▼
[2단계: 공격 종합]
    │  치명적 약점 vs 사소한 우려 분류
    │  공통 지적 사항 하이라이트
    ▼
[3단계: 반박 유도]
    │  "이 공격에 어떻게 답하시겠어요?"
    ▼
저장: research/[TICKER]/challenge_YYYY-MM-DD.md
```

## 페르소나별 공격 포인트

### 세상학개론 (내러티브 + 유동성)

```markdown
공격 관점:
- "이게 진짜 '시대의 병목'인가? 그냥 유행 아닌가?"
- "내러티브가 이미 주가에 반영된 거 아닌가?"
- "유동성 환경이 바뀌면 이 스토리 무너지는 거 아닌가?"
- "피지컬 AI / 소버린 AI 트렌드와 얼마나 연결되어 있나?"

핵심 질문: "내러티브 프리미엄이 과도한 거 아닌가?"
```

### Cathie Wood (파괴적 혁신)

```markdown
공격 관점:
- "이 기업이 정말 '파괴적'인가? 점진적 개선 아닌가?"
- "기존 강자들이 이 시장에 들어오면 버틸 수 있나?"
- "5년 후에 이 기술이 commodity화되면?"
- "혁신 속도가 너무 느린 거 아닌가?"

핵심 질문: "진짜 세상을 바꿀 기업인가, 그냥 좋은 기업인가?"
```

### Michael Burry (역발상 + Deep Value)

```markdown
공격 관점:
- "이거 버블 아닌가? 모두가 좋다고 할 때가 위험하다"
- "이 밸류에이션 정당화할 수 있나? 숫자로 보여달라"
- "시장이 뭔가 놓치고 있다고? 그게 착각 아닌가?"
- "다들 AI AI 할 때 진짜 수혜주가 맞나?"
- "경영진이 주주가치보다 성장에 집착하는 거 아닌가?"

핵심 질문: "남들이 다 사는 걸 따라 사는 거 아닌가?"
```

### Terry Smith (품질 복리)

```markdown
공격 관점:
- "10년 보유할 수 있는 기업인가?"
- "ROE가 지속적으로 높은가? 왜?"
- "이 마진이 유지될 수 있나? 경쟁 심화되면?"
- "경영진 퀄리티가 검증되었나?"
- "부채 수준이 위험하지 않나?"

핵심 질문: "복리로 성장할 퀄리티가 있나, 아니면 한방 노리는 건가?"
```

### Howard Marks (사이클 + 리스크)

```markdown
공격 관점:
- "지금 사이클 어디인가? 정상에서 사는 거 아닌가?"
- "이 섹터에 돈이 너무 몰린 거 아닌가?"
- "시장 심리가 과열된 건 아닌가?"
- "리스크 대비 보상이 적절한가?"
- "모두가 낙관적일 때 경계해야 하는 거 아닌가?"

핵심 질문: "좋은 기업이지만 좋은 가격인가?"
```

### Stanley Druckenmiller (매크로 + 타이밍)

```markdown
공격 관점:
- "금리 인상/인하가 이 기업에 어떤 영향?"
- "달러 강세/약세 시나리오는?"
- "경기 침체 오면 이 기업 실적 얼마나 빠지나?"
- "지금 진입 타이밍이 맞나? 더 좋은 가격 올 수 있지 않나?"
- "매크로 헤드윈드를 과소평가하는 거 아닌가?"

핵심 질문: "타이밍이 틀리면 3년 물려있을 각오 있나?"
```

### Bill Ackman (행동주의 + 촉매)

```markdown
공격 관점:
- "가치를 끌어올릴 촉매가 뭔가? 구체적으로"
- "경영진이 주주가치 극대화에 관심 있나?"
- "이 주가가 오르려면 뭐가 바뀌어야 하나?"
- "숨겨진 가치가 있다면 왜 아직 반영 안 됐나?"
- "내가 대주주라면 뭘 바꾸고 싶나?"

핵심 질문: "이 주식이 오를 '이유'가 뭔가? 그냥 '희망' 아닌가?"
```
