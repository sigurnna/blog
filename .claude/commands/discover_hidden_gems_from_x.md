# /discover-gems

> X(트위터) 기반 유망 종목 발굴 커맨드

## 목표

X에서 노이즈를 제거하고, 투자자 프로필에 부합하는 **10배 성장 잠재력**을 가진 숨겨진 종목을 발굴한다.

## 신뢰할 수 있는 소스

| 유형 | 설명 |
|------|------|
| 산업 전문가 | AI, 우주, 바이오 분야 현직 엔지니어/연구원 |
| 검증된 분석가 | 펀더멘탈과 구조적 변화를 논하는 계정 |
| 기관급 인사이트 | 헤지펀드 매니저, VC 등 거대 자금 흐름을 읽는 사람들 |

## 필터링 기준

### 내러티브 적합성
- **거대 내러티브**: 현재 시장 주도 트렌드(AI 패권, 우주 산업 등)와 연결되는가?
- **유동성 수혜**: 정책적 지원이나 자금 유입이 기대되는 분야인가?

### 종목 검증
1. **Thesis 확인**: "왜 지금인가?"에 대한 명확한 논리가 있는가?
2. **초기 발굴**: 대중에게 덜 알려졌거나 오해받는 종목인가?
3. **10배 가능성**: 3~5년 내 10배 수익 가능한 비즈니스 모델인가?

## 출력 형식

```markdown
## [발굴 종목] 기업명 (티커)

### 소스 및 핵심 논리
- **소스**: (주요 계정 및 게시글)
- **핵심 논리**: (3줄 이내 요약)

### 내러티브 부합성
- 연결된 거대 테마
- 주요 키워드 연관성

### 검증 의견
| 항목 | 내용 |
|------|------|
| 기회 | (성장 잠재력) |
| 리스크 | (확인 필요한 위험 요소) |
| 다음 단계 | 심층 분석 / 관망 / 매수 고려 |
```
