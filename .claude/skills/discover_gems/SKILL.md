# /discover-gems

> X(트위터) 기반 유망 종목 발굴

## 목표

X에서 노이즈를 제거하고, 투자자 프로필에 부합하는 **10배 성장 잠재력**을 가진 숨겨진 종목을 발굴한다.

---

## 신뢰할 수 있는 소스

| 유형 | 설명 |
|------|------|
| 산업 전문가 | AI, 우주, 바이오 분야 현직 엔지니어/연구원 |
| 검증된 분석가 | 펀더멘탈과 구조적 변화를 논하는 계정 |
| 기관급 인사이트 | 헤지펀드 매니저, VC 등 거대 자금 흐름을 읽는 사람들 |

### 회피 대상

- 펌프앤덤프 계정
- 팔로워 수만 많고 실적 없는 인플루언서
- 감정적/선정적 헤드라인만 생산하는 계정

---

## 필터링 기준

### 1. 내러티브 적합성

| 기준 | 질문 |
|------|------|
| 거대 내러티브 | 현재 시장 주도 트렌드(AI 패권, 우주 산업 등)와 연결되는가? |
| 유동성 수혜 | 정책적 지원이나 자금 유입이 기대되는 분야인가? |
| 시대적 병목 | 해결하면 큰 가치를 창출하는 문제와 연결되는가? |

### 2. 종목 검증

| 기준 | 질문 |
|------|------|
| Thesis 확인 | "왜 지금인가?"에 대한 명확한 논리가 있는가? |
| 초기 발굴 | 대중에게 덜 알려졌거나 오해받는 종목인가? |
| 10배 가능성 | 3~5년 내 10배 수익 가능한 비즈니스 모델인가? |

```
    └─ 필터링 기준 적용
       - 내러티브 적합성
       - 10배 가능성
              │
[4단계: 결과 정리]
    └─ 출력 형식에 맞춰 정리
```
