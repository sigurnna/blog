## Executive Summary

두나무는 암호화폐 버블이 절정에 달했을 때 가장 위험한 비즈니스 모델이다. 매출의 97%가 거래 수수료에 의존하는 구조는 2021년 3.7조원에서 2022년 1.2조원으로 67% 급락한 과거가 이를 증명한다. 현재 장외주가 34만원(시총 11.9조)은 PER 12.1x로 저평가처럼 보이지만, 암호화폐 시장이 정상화되면 실적은 다시 2022년 수준으로 회귀할 것이다. 안전마진이 없다.

2. **사이클 최상단**: 비트코인이 2025년 10월 ATH 기록 후 현재 조정 중($85K~$89K). 거래량 급증기의 실적으로 밸류에이션 산정하면 함정에 빠진다
3. **장외주가 52주 고저**: 105,000원 ~ 419,000원 (4배 변동성). 현재 34만원은 중간값에 불과

- Coinbase 대비 저평가처럼 보이지만, Coinbase는 나스닥 상장사로 유동성 프리미엄이 있고 미국 시장의 규제 진입장벽이 한국보다 훨씬 높다

## 역발상 기회

### 시장이 과소평가하는 것

1. **없음 - 과대평가가 지배적**

### 시장이 과대평가하는 것

1. **해자의 지속성**
   - 시장점유율: 2023년 90% → 2024년 초 70% → 2025.12 59.4%
   - 빗썸의 수수료 무료 정책으로 불과 2년 만에 30%p 점유율 상실
   - 케이뱅크 독점 연동도 법인 투자 허용(2025 하반기~)으로 희석 가능성
   - **Burry의 분석**: "독점적 지위"라는 내러티브는 이미 무너지고 있다. 재무제표가 말해준다

2. **"제도권 편입" 내러티브**
   - 비트코인 ETF 승인 = 시장 성숙 아니다
   - 서브프라임 모기지도 AAA 등급을 받았다
   - 규제는 위험을 줄이지 못했다. 오히려 진입장벽만 높였을 뿐
   - **Burry의 경고**: "가상자산이용자보호법 시행"이 시장 안전성을 보장하지 않는다

3. **네이버 합병 기업가치 40~50조원**
   - 기업가치는 현금흐름으로 결정된다
   - 2024년 순이익 9,838억원을 정규화하면? 2022년(1,308억원) / 2023년(8,050억원) / 2024년(9,838억원)
   - 암호화폐 시장 사이클을 고려한 정규화 순이익은 약 3,000~5,000억원 수준
   - 정규화 PER 15x 적용 시 적정 기업가치: 4.5조~7.5조원
   - **Burry의 판단**: 현재 장외시총 11.9조는 이미 과대평가. 40~50조원은 망상이다

4. **"한국 인구 20%가 거래"의 의미**
   - 대중화 = 기회 아니다. 대중화 = 천정이다
   - 2021년 누적 회원 1,000만명 돌파 후 매출 67% 급락(3.7조 → 1.2조)
   - 2025년 누적 회원 1,326만명. 추가 성장 여력은 제한적
   - **Burry의 경고**: 모든 사람이 같은 방향으로 달릴 때가 가장 위험하다

## 안전마진 계산

| 항목 | 수치 |
|------|------|
| **정규화 순이익** | 3,000~5,000억원 (2022~2024 평균, 사이클 조정) |
| **적정 PER** | 15x (사이클 기업 할인) |
| **내재가치 추정** | 4.5조~7.5조원 |
| **현재 장외시총** | 11.9조원 (주가 342,000원) |
| **안전마진** | **-37% ~ -59%** (과대평가) |

### 계산 근거

1. **정규화 순이익 산정**:
   - 2021년: 2조 2,177억원 (버블 피크 → 제외)
   - 2022년: 1,308억원 (크립토 윈터 → 제외)
   - 2023년: 8,050억원
   - 2024년: 9,838억원 (비트코인 ATH $126K 효과)
   - **정규화 접근**: 2023~2024 평균 = 8,944억원
   - **보수적 할인 50%**: 약 4,000~5,000억원 (암호화폐 사이클 하락기 시나리오)

2. **PER 15x 적용 근거**:
   - 코인베이스(NASDAQ): PER 30x+, 하지만 미국 1위 + 상장사 프리미엄
   - 두나무: 비상장 + 시장점유율 하락 + 극심한 실적 변동성 → 할인 필요
   - Lynch 기준: Cyclical + Fast Grower → PER 15x는 적정

3. **내재가치 범위**:
   - Bull Case (정규화 순이익 5,000억 × PER 15x): 7.5조원
   - Base Case (정규화 순이익 4,000억 × PER 15x): 6조원
   - Bear Case (정규화 순이익 3,000억 × PER 15x): 4.5조원

### 최악의 시나리오

**시나리오**: 비트코인이 2026~2027년 $30,000 수준으로 재차 하락 (2022년 크립토 윈터 재현)

| 항목 | 수치 |
|------|------|
| 매출 | 5,000억원 (2024년 대비 -71%, 2022년 수준) |
| 영업이익률 | 50% (고정비 증가로 마진 압박) |
| 순이익 | 1,500억원 |
| PER | 10x (극심한 불확실성 할인) |
| 기업가치 | **1.5조원** |
| 현재 대비 손실 | **-87%** |

**Burry의 결론**: 안전마진이 없다. 오히려 현재 가격 자체가 최악의 시나리오 대비 7~8배 고평가됐다.

| **권장 액션** | **패스 / 숏 고려** |

### 판단 근거

1. **매출의 97%가 거래 수수료 - 극도로 취약한 비즈니스 모델**
   - 2021년 3.7조 → 2022년 1.2조 (-67%). 이미 한 번 붕괴했다
   - 비트코인 가격과 1:1 연동. 헤지 불가능한 구조
   - **Burry의 평가**: "이건 비즈니스가 아니다. 도박장 수수료다"

2. **시장점유율 급락 - 해자 붕괴 중**
   - 90% → 60% (2년 만에 30%p 상실)
   - 빗썸의 수수료 무료 정책에 무방비
   - 바이낸스 국내 진출(2025.10 고팍스 인수) → 추가 위협
   - **Burry의 관점**: "독점 프리미엄은 사라지고 있다. 영업이익률 68.5%는 지속 불가능하다"

3. **"이번엔 다르다" 내러티브가 지배적 - 전형적인 버블 신호**
   - "비트코인 ETF로 제도권 편입"
   - "한국 인구 20%가 거래. 대중화 완료"
   - "네이버 합병으로 40~50조 기업가치"
   - **Burry의 경고**: 2007년에도 모두가 "부동산은 안전하다"고 했다. 군중이 열광할 때가 가장 위험하다

4. **밸류에이션 함정 - 정규화되지 않은 수익 기반**
   - PER 12.1x는 저평가처럼 보이지만, 2024년 순이익 9,838억원은 비트코인 ATH의 결과물
   - 정규화 순이익 4,000억 기준 → PER 29.8x (과대평가)
   - 크립토 윈터 시나리오 → PER 79x (극심한 과대평가)
   - **Burry의 결론**: "재무제표가 말해주는 건 다르다. 지금은 팔 때다"

5. **ROE 변동성 - Buffett 기준 부적합**
   - 2021: 68.7% → 2022: 4.2% → 2023: 20% → 2024: 20.1%
   - 표준편차가 극심함. 안정적 현금흐름 기대 불가
   - **Burry의 평가**: "이런 기업은 10년 보유할 수 없다. 사이클 톱에서 팔고 바닥에서 사야 한다"

6. **현금 보유 제한적 - 유동성 리스크**
   - 실질 현금 3,500억원 (시총 11.9조 대비 2.9%)
   - 고객 예치금 8조 532억원은 별도 관리 (사용 불가)
   - 시장 급락 시 배당/자사주 매입 여력 부족

7. **안전마진 부재 - 현재가 이미 고평가**
   - 내재가치 4.5~7.5조 vs 현재 시총 11.9조
   - 최악의 시나리오(크립토 윈터) 시 기업가치 1.5조 → 87% 손실
   - **Burry의 결론**: "매수 시점이 아니다. 숏 포지션을 고려할 때다"
