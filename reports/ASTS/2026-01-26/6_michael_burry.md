# AST SpaceMobile (ASTS) 투자 분석 - Michael Burry 관점

> 분석일: 2026-01-26
> 분석 관점: Michael Burry (역발상 + Deep Value)

---

## Executive Summary

시가총액 $42B에 TTM 매출 $18.5M, P/S 2,269배. 재무제표가 말해주는 건 단 하나다: **이건 밸류에이션이 아니라 신앙이다.** 시장은 "우주 기반 셀룰러 혁명"이라는 내러티브에 열광하고 있지만, 안전마진은 존재하지 않는다. 현금은 3분기면 소진되고, SpaceX는 이미 650기 위성으로 앞서 나갔다. 이건 패스다.

## 버블 체크

### 밸류에이션 분석

**현재 밸류에이션**:

| 지표 | 값 | 해석 |
|------|-----|------|
| 시가총액 | $42.05B | |
| TTM 매출 | $18.53M | |
| P/S | 2,269x | **Dot-com 버블 수준** |
| P/B | 24.9x | |
| EV/Revenue | 2,243x | |
| Owner Earnings | -$1,010M (TTM) | 대규모 마이너스 |

**역사적 비교**:
- Dot-com 버블 당시 Pets.com: P/S 약 30-50배 (이미 매출 있었음)
- ASTS의 P/S 2,269배는 **비교 불가능한 수준**
- 이건 기업이 아니라 **옵션**이다

**동종 업계 대비**:

| 기업 | 매출 (TTM) | 위성 수 | 서비스 현황 |
|------|-----------|---------|-----------|
| SpaceX Starlink | $11.8B (전체) | 650+ D2C / 10,000+ 전체 | 문자 서비스 상용화 (2025.07) |
| ASTS | $18.5M | 5기 | 상용 전 (2026 예정) |

SpaceX는 이미 650기를 운영 중이고 상용 서비스를 시작했다. ASTS는 5기로 아직 베타조차 시작 안 했다. 시장은 이 격차를 무시하고 있다.

### 군중 심리 체크

**시장 컨센서스**:
- "우주 기반 셀룰러는 조 단위 시장이다"
- "MNO 50개사 파트너십이 확실한 수요를 증명한다"
- "SpaceX와 달리 음성/데이터/영상을 지원한다"
- "정부 계약이 기술력을 입증했다"

**군중의 방향**:
- 공매도 비율: 10.28% (38.05M주) - 상당수지만 압도적이진 않음
- 기관 보유: 36.87%
- 개인 투자자: 20.88%
- 52주 최고가: $120.93 (2026-01-21, 불과 5일 전)

모두가 같은 방향을 보고 있다. **"이번엔 다르다"는 주장이 도처에 깔려 있다.**

### Red Flags

- [x] **"이번엔 다르다" 논리가 있는가?**
  - "기존 위성통신과 달리 일반 스마트폰을 직접 지원한다"
  - "Block 1 위성 하나가 Starlink D2C 100배 용량이다"
  - "MNO 파트너십이 수요를 보장한다"

- [x] **레버리지가 과도한가?**
  - 직접적 부채: D/E 44% (낮음)
  - 하지만 **운영 레버리지**는 극단적:
    - 분기당 $363M 현금 소진
    - 현금 $1.2B, 약 3분기 런웨이
    - 추가 자금 조달 불가피 (주식 희석 가속화)

- [x] **신규 투자자가 대거 유입 중인가?**
  - 5년간 발행주식수 5배 증가 (54M → 370M)
  - 2025년에만 75% 증가 (155M → 273M)
  - 52주 최고가가 **5일 전**

**결론: Red Flag 3개 모두 점등. 전형적인 버블 신호.**

## 역발상 기회

### 시장이 과소평가하는 것

**없다.** 역발상 투자는 군중이 과도하게 비관할 때 기회가 온다. 지금은 그 반대다.

유일하게 주목할 점:
1. **특허 3,700+건**: 실제 기술 자산이 존재하긴 함
2. **MNO 파트너십**: 50개 통신사는 실제 관계다 (돈을 얼마나 낼지는 별개 문제지만)
3. **정부 계약**: MDA SHIELD, SDA $43M - 듀얼유스 가치는 인정받음

하지만 이 모든 건 이미 $42B 시가총액에 **과도하게** 반영되었다.

### 시장이 과대평가하는 것

1. **"First Mover Advantage" 신화**
   - SpaceX가 650기로 이미 상용 서비스 중
   - ASTS는 5기로 아직 베타 전
   - "먼저 도착한 자"는 SpaceX다

2. **"기술 우위" 환상**
   - Block 1 위성이 "100배 용량"? 그래서 뭐?
   - SpaceX는 양(數)으로 이미 이겼다
   - 위성 5기 vs 650기 - 커버리지 싸움은 이미 끝났다

3. **수익 전환 타이밍 낙관론**
   - 시장은 2026년 상용화를 확정된 것처럼 가격에 반영 중
   - 하지만 현실: 현금 3분기, 추가 자금 조달 불가피
   - 베타 서비스에 필요한 25기조차 아직 멀었다 (현재 5기)

4. **"$1T TAM" 허상**
   - 회사가 주장하는 TAM $1T는 **이론적 최대치**
   - 실제 Direct-to-Phone 수익 (2033): $42B (Novaspace)
   - D2D 시장 (2033): $15B (Allied Market Research)
   - $1T는 마케팅 숫자다

5. **"MNO 파트너십 = 확정 수익" 착각**
   - 파트너십은 "관심 표명"이지 "확정 발주"가 아니다
   - 실제 계약 커밋먼트: $1B+ (구체적 금액/타이밍 불명확)
   - stc $175M 선불은 10년 계약 - 연 $17.5M에 불과

## 안전마진 계산

| 항목 | 수치 |
|------|------|
| 현재 주가 | $113.55 (2026-01-26) |
| 시가총액 | $42.05B |
| Book Value Per Share | $4.56 |
| P/B | 24.9x |
| **안전마진** | **없음** |

### 내재가치 추정 시도

**전통적 DCF는 불가능**:
- FCF -$916M (TTM)
- 순이익 -$304M (TTM)
- Owner Earnings: 대규모 마이너스

**Asset Play 관점**:

| 자산 | 가치 추정 |
|------|----------|
| 현금 | $1.2B |
| 위성 6기 | $100-150M (건당 $16-22M) |
| 특허 3,700+건 | 추정 불가 (시장가치 불명확) |
| 제조 시설 | $100-200M |
| **총 유형자산** | **~$1.5-1.6B** |

시가총액 $42B vs 유형자산 $1.6B = **26배 프리미엄**

**이건 기업이 아니라 콜옵션이다:**
- 내재가치: 추정 불가
- 안전마진: **없음**
- 가격은 "미래 스토리"로만 정당화됨

### 최악의 시나리오

**시나리오 1: 자금 조달 실패**
- 현금 소진 (3분기 후)
- 추가 지분 희석 또는 파산
- 주가: $4.56 (Book Value) 이하로 하락 가능
- **손실: -96%**

**시나리오 2: SpaceX 지배력 확대**
- Starlink D2C가 시장 선점 (이미 진행 중)
- ASTS는 "기술력 있는 2등" 포지션
- 시장은 "승자독식" 패턴 가능
- 밸류에이션 재조정: P/S 2,269x → 50x (그래도 과하지만)
- 주가: ~$2.50 수준
- **손실: -98%**

**시나리오 3: 상용화 지연**
- 2026년 목표 미달성
- 추가 1-2년 현금 소진
- 주식 희석 가속화 (현재 370M → 700M+)
- 주가: $30-50 범위
- **손실: -56% ~ -74%**

**최선의 시나리오조차 위험하다:**
- 상용화 성공, 2027년 ROE +1.8% (BofA 전망)
- 하지만 그때까지 주식 희석으로 주당 가치 반토막
- 매출 $267M (2026 전망)으로도 P/S는 여전히 150배+

## 투자 판단

### 최종 판단

| 항목 | 결론 |
|------|------|
| **버블 여부** | **버블** (P/S 2,269배, 군중 열광, Red Flag 3개) |
| **안전마진** | **없음** (내재가치 추정 불가, 하방은 -96%) |
| **확신도** | ★★★★★ (패스 확신) |
| **권장 액션** | **패스 / 숏 고려** |

### 판단 근거

1. **밸류에이션이 말이 안 된다**
   - P/S 2,269배는 Dot-com 버블도 울고 갈 수준
   - 시가총액 $42B에 매출 $18M - 이건 가격 발견이 아니라 신앙 고백이다
   - Owner Earnings -$1B, FCF -$916M - 돈을 버는 게 아니라 태우는 기업

2. **SpaceX가 이미 이겼다**
   - 650기 vs 5기 - 게임은 이미 끝났다
   - Starlink는 2025년 7월 상용화, ASTS는 2026년 "목표"
   - EchoStar $17B 인수로 수직 통합 완성
   - ASTS의 "기술 우위" 주장은 **숫자 앞에서 무력하다**

3. **현금 런웨이 3분기 - 희석의 악순환**
   - 현금 $1.2B, 분기당 $363M 소진
   - 60기 발사 비용 $1.3B - 현재 현금으로 부족
   - 추가 자금 조달 불가피 = 주식 희석 가속화
   - 5년간 이미 5배 희석 (54M → 370M)
   - **희석은 "조용한 파산"이다**

4. **"이번엔 다르다" 신호 점멸**
   - 모두가 "우주 셀룰러 혁명"을 외치고 있다
   - MNO 파트너십을 "확정 수익"처럼 포장
   - 특허 3,700건을 "해자"로 착각
   - 52주 최고가가 **5일 전** - 전형적인 광기의 막바지

5. **안전마진이 존재하지 않는다**
   - 내재가치 추정 불가 (음수 수익, 음수 FCF)
   - 유형자산 $1.6B vs 시가총액 $42B = 26배 프리미엄
   - 최악의 시나리오: -96% 손실
   - Benjamin Graham이 보면 기절할 구조

### Burry의 조언

**이건 투자가 아니라 투기다.**

시장은 ASTS를 "Next Big Thing"으로 포장하고 있지만, 재무제표는 다른 이야기를 한다:
- 5년간 지속 적자
- 연간 $300M+ 현금 소진
- P/S 2,269배
- 경쟁자는 이미 650기로 상용화 중

**군중이 열광할 때 나는 도망간다.** 이건 2000년 Pets.com의 재연이다. 차이가 있다면 Pets.com은 적어도 P/S가 50배였다는 것뿐이다.

안전마진 없이 투자하는 건 **금융적 자살**이다. ASTS는 언젠가 "기술력 있는 기업"이 될 수도 있다. 하지만 그건 주가 $5 이하에서 고민할 문제다.

**지금 이 가격에서? 절대 안 된다.**

## 추가 고려사항

### 만약 숏을 친다면 주의할 점

1. **공매도 비용**: 10% 공매도 비율, 차입 비용 높을 수 있음
2. **타이밍 리스크**: "시장은 당신이 파산할 때까지 비이성적일 수 있다"
3. **촉매 이벤트**:
   - 2026년 상용화 지연 발표
   - 추가 자금 조달 발표 (희석 우려)
   - SpaceX Starlink D2C 확대 발표
   - 분기 실적 (현금 소진 속도)

### 언제 다시 볼 것인가?

**재진입 조건**:
- 주가 $10 이하 (P/B 2배 이하)
- 상용 서비스 실제 런칭 + 분기 매출 $50M+ 달성
- SpaceX와의 시장 포지션 명확화
- 추가 자금 조달 완료 (2년+ 런웨이 확보)

그때 가서도 **안전마진**을 먼저 계산할 것이다.
