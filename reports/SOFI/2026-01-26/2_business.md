## 비즈니스 모델 팩트시트: SoFi Technologies (SOFI)

> 기준일: 2026-01-26

### 사업 개요

| 항목 | 내용 |
|------|------|
| 핵심 사업 | 통합 금융 서비스 플랫폼 - 대출, 뱅킹, 투자, 금융기술 인프라를 단일 앱에서 제공하는 디지털 금융 기업 |
| 수익 모델 | 순이자수익(NII), 대출 수수료, 거래 수수료(Interchange), 플랫폼 이용료(BaaS), 대출 플랫폼 수수료 |
| 주요 고객 | B2C (12.6M 개인 회원), B2B (Galileo 플랫폼 고객사 - 금융기관, 핀테크) |

### 주요 제품/서비스

| 제품/서비스 | 설명 |
|-------------|------|
| 개인 대출 | Q3 2025 $7.5B 실행 (YoY +53%) |
| 학자금 대출 | Q3 2025 $1.5B 실행 (YoY +58%) |
| 주택 대출 | Q3 2025 $945M 실행 (주택담보대출 $352M 포함) |
| SoFi Money | 예금 상품 - $32.9B 예금 (Q3 2025) |
| SoFi Invest | 주식/ETF/옵션/로보어드바이저 |
| SoFi Relay | 금융 데이터 통합 서비스 |
| SoFi Credit Card | 신용카드 |
| Galileo | 금융기술 플랫폼 (카드 발급, 결제 처리, API) |
| Technisys | 클라우드 기반 코어뱅킹 시스템 |

### 확장성 구조

| 항목 | 내용 |
|------|------|
| 한계 비용 | 디지털 전용 모델로 지점 비용 없음. 예금 기반 대출로 Warehouse 대비 190bp 저렴한 자금 조달 |
| 규모의 경제 | 있음 - 회원 증가 시 크로스셀 통해 CAC 없이 추가 수익 창출. 예금 증가로 NIM 개선 |
| 플랫폼/네트워크 효과 | 있음 - 양면 플랫폼 (Galileo: 더 많은 파트너 = 더 많은 거래량 = 더 낮은 단위 비용) |
| 지리적 확장 | 미국 중심. Galileo/Technisys 통해 중남미 일부 진출 |
