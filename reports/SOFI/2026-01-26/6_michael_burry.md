# SoFi Technologies 투자 분석 - Michael Burry 관점

> 분석일: 2026-01-26
> 분석 관점: Michael Burry (역발상 + Deep Value)

## Executive Summary

**이건 Fintech 버블 2.0이다.** P/E 48배에 거래되는 기업이 ROE 8%, ROIC 2%를 기록하고 있다. 시장은 "성장 스토리"에 취해 있지만, 재무제표는 다른 이야기를 한다: 연간 FCF -$3B, 주식 희석 연 17%, 5년간 누적 적자. 2024년 겨우 흑자 전환했다고 모두가 환호하지만, 이건 금리 사이클의 운일 뿐이다. 금리가 다시 오르면? 역사는 반복된다.

## 버블 체크

### 밸류에이션 분석

| 지표 | SoFi | 섹터 평균 | 프리미엄 |
|------|------|----------|---------|
| **P/E (TTM)** | 47.69 | 11.54 (Financial Services) | **+313%** |
| **P/B** | 3.55 | 1.05 (Credit Services) | **+238%** |
| **P/S** | 8.77 | - | - |
| **EV/Sales** | 9.79 | - | - |

**현재 밸류에이션 vs 역사적 평균**:
- 금융 서비스 기업 평균 P/E의 4배
- Credit Services 업계 평균 P/B의 3배 이상
- 성장률 조정 PEG 0.88은 "저평가"처럼 보이지만, 이건 함정이다

**문제의 본질**:
1. **일시적 흑자를 구조적 수익성으로 착각하고 있다**
2. 2024년 첫 연간 흑자 = 8분기 (겨우 2년)
3. 그 전 5년간? ROE 평균 **-4.89%**
4. 2025년 성장률로 밸류에이션을 정당화하지만, 성장은 **주식 희석과 부채로 산 것**

### 군중 심리 체크

**시장 컨센서스**:
- "원스톱 금융 슈퍼앱"
- "디지털 뱅킹의 미래"
- "은행 차터로 해자 구축 완료"
- "Galileo로 B2B 플랫폼 수익 다각화"

**모두가 같은 방향을 보고 있는가?** ✅ 그렇다.

증거:
- 52주 수익률 +88%
- 시가총액 $32.67B (전통 은행들과 비교할 수준)
- Beta 1.93 (시장 대비 2배 변동성 = 투기적 베팅)
- 2025년 12월 $1.5B 유상증자 성공 (주당 $27.50에 5% 희석)

**내러티브 분석**:
1. "Fintech 혁신" → 2019-2021년에도 똑같이 들었다
2. "크로스셀로 해자" → 1.48개 상품/회원 = 그냥 기본적인 뱅킹 번들링
3. "은행 차터로 비용 우위" → 170bp 절감 = 좋지만, 이건 이미 주가에 반영됨
4. "40% CAGR 회원 성장" → 그런데 왜 ROE는 8%?

### Red Flags

- [x] **"이번엔 다르다" 논리가 있는가?**
  - "우린 네오뱅크가 아니라 Financial Services Technology Company다" ← 전형적인 버블 화법
  - "Galileo B2B 플랫폼으로 경기방어적이다" ← 그런데 Q3 2025 계정 성장률 -1%

- [x] **레버리지가 과도한가?**
  - Debt/Equity 0.31x = 낮아 보이지만
  - 총부채 $36.5B / 자기자본 $8.8B
  - **대출 포트폴리오 자체가 레버리지다** (무담보 개인 대출 비중 높음)

- [x] **신규 투자자가 대거 유입 중인가?**
  - 52주 +88% 수익률
  - 2025년 12월 $1.5B 유상증자 성공
  - **밸류에이션이 정당화되려면 "영원한 성장"이 필요한 수준**

- [x] **주식 희석이 지속되는가?**
  - 2022: 934M 주 → 2025 Q3: 1.29B 주 (**+38% 증가**)
  - 최근 3년 연평균 **+12-17% 희석**
  - 성장의 대가: 기존 주주 지분 희석

## 역발상 기회

### 시장이 과소평가하는 것

1. **금리 사이클 리스크**
   - 2024년 흑자 전환 = 금리 하락 사이클 시작과 정확히 일치
   - Fed가 금리를 75bp 인하 → 대출 수요 증가 → 일시적 수익성 개선
   - 하지만 NIM(순이자마진) 압박은 이미 시작됨 (Q3 5.84%, Q2 5.86%)

2. **신용 리스크 잠재력**
   - 무담보 개인 대출이 핵심 사업
   - Q3 2025 개인 대출 실행 $7.5B (YoY +53%)
   - 경기 둔화 시 연체율 급등 가능성 → 시장은 "낙관 시나리오"만 프라이싱

3. **FCF 적자의 구조적 문제**
   - TTM FCF -$3.03B
   - 영업현금흐름 -$3.46B
   - "대출 증가는 성장의 증거"라는 내러티브 ← **현금 소진의 증거이기도 하다**

### 시장이 과대평가하는 것

1. **"해자" 내러티브**
   - 5개 해자 보유? 재무제표는 거짓말하지 않는다:
     - ROIC 2.27% ← 해자가 있다면 최소 15% 이상이어야 한다
     - ROE 8.26% ← Buffett 기준 15% 미달
   - 진짜 해자가 있다면, 왜 계속 유상증자로 자금을 조달하는가?

2. **"Galileo B2B 플랫폼" 가치**
   - 1.58억 계정 지원 ← 인상적으로 들리지만
   - Q3 2025 성장률 **-1% YoY** ← 성장이 멈췄다
   - Technology Platform 매출 $114.6M (Q3) ← 전체 매출 $962M의 12%에 불과

3. **"영구적 성장" 가정**
   - 회원 +35% YoY, 상품 +36% YoY
   - 하지만 주식수도 +17% YoY 증가
   - **희석 조정 성장률은 훨씬 낮다**
   - TAM 스토리: "$653B by 2030" ← 모든 버블이 쓰는 수법

## 안전마진 계산

### 내재가치 추정 (Reverse DCF)

**현재 주가 정당화를 위한 가정**:
- 주가: $26 (2026년 1월 기준)
- 시가총액: $32.67B
- 필요한 미래 FCF 성장률: 연 30%+를 10년 이상 유지

**현실 체크**:
1. 현재 FCF: **-$3.03B (마이너스)**
2. FCF를 양전환하려면?
   - 대출 증가 속도 둔화 필요 ← 성장 포기
   - 또는 수익성 급격히 개선 ← 영업이익률 12% → 30%+로?
3. **두 가지 모두 비현실적**

| 항목 | 수치 |
|------|------|
| 현재 가격 | $26.00 |
| 내재가치 추정 (보수적) | $8-12 |
| 내재가치 추정 (비관적) | $5-8 |
| **안전마진** | **-55% ~ -70% (음수)** |

### 밸류 트랩 체크

**Lynch 기준**: 3/4 충족 ← Fast Grower로 분류
**Buffett 기준**: 0/4 충족 ← **투자 부적격**
**Fisher 기준**: 1/3 충족 ← 주식 희석 지속
**Munger 기준**: 2/4 충족 ← ROIC 2.27% (기준 15% 미달)

**종합 충족률**: 6/15 (40%) ← **가치투자 기준으로 실격**

### 최악의 시나리오

**만약 금리가 다시 오르면?**
1. 대출 수요 급락 → 회원 성장 둔화
2. NIM 압박 심화 → 수익성 악화
3. 신용 리스크 현실화 → 대손충당금 증가
4. 주가 폭락 → 유상증자 불가능 → 현금 소진

**최악의 경우 얼마나 잃는가?**
- 현재가 $26 기준
- 하방: **$5-8 (70-80% 하락)**
- 근거: P/B 3.55 → 1.0으로 회귀 시 -72%
- 역사적 선례: 2022년 고점 대비 SoFi 자체가 -75% 하락 경험

**상방은?**
- 최선의 시나리오: $35-40 (+35-54%)
- 리스크/리워드 비율: **3:1 (하방 편향)**

## 투자 판단

### 최종 판단

| 항목 | 결론 |
|------|------|
| **버블 여부** | **과열** (섹터 평균 대비 P/E +313%, P/B +238%) |
| **안전마진** | **없음** (음수 안전마진 -55% ~ -70%) |
| **확신도** | ★★★★☆ (숏 포지션에 대한 확신) |
| **권장 액션** | **숏 고려** (또는 최소한 패스) |

### 판단 근거

1. **밸류에이션 버블**
   - P/E 48배는 금융주에게 정당화 불가능
   - 5년 평균 ROE -4.89%, 현재 ROE 8.26% = 구조적 수익성 미증명
   - "성장"은 주식 희석 +17% YoY로 산 것

2. **재무제표가 말하는 진실**
   - FCF -$3B = 현금 소각 기계
   - ROIC 2.27% = 해자 없음의 증거
   - 부채/순이익 5.6년 = 레버리지 부담

3. **군중 심리 경고**
   - 모든 사람이 "디지털 뱅킹의 미래"를 외치고 있다
   - 52주 +88% 수익률 = 과열 신호
   - "이번엔 다르다" 내러티브 만연

4. **금리 사이클 리스크**
   - 2024년 흑자 전환 = 금리 하락 사이클의 산물
   - Fed 금리 동결/재인상 시 모든 내러티브 붕괴
   - NIM 이미 하락 시작 (5.86% → 5.84%)

5. **비교 우위 없음**
   - Chime: 2,000만 회원 (SoFi의 1.6배)
   - Robinhood: 시총 $122B (SoFi의 3.7배)
   - JPMorgan: 디지털 뱅킹 강화 중
   - **SoFi의 차별점? 없다. "다 하는" 것은 "다 평범하다"와 동의어**

### 역발상 포지션

**Short Thesis**:
1. P/E 48배 → 15배로 회귀 시 -69% 하락
2. 금리 재인상 시 NIM 붕괴 → 적자 재전환 가능
3. 경기 둔화 시 무담보 대출 연체율 급등
4. 더 이상 유상증자 불가능 → 성장 동력 상실

**진입 타이밍**:
- 즉시 (현 $26 수준)
- 또는 다음 분기 실적 발표 후 (NIM 하락 가속화 확인 시)

**목표가**: $8-12 (70-54% 하락)

**손절**: $32 이상 돌파 시 (+23%)

### 반대 의견에 대한 반박

**"하지만 회원이 35% 성장하고 있는데?"**
→ 주식도 17% 희석되고 있다. 희석 조정 성장률은 18%에 불과. 그리고 성장은 영원하지 않다.

**"은행 차터로 비용 우위가 있는데?"**
→ 170bp 절감은 좋지만, 이미 주가에 반영됨. 그리고 ROIC 2%면 "우위" 아니다.

**"Galileo B2B 플랫폼이 있는데?"**
→ 성장률 -1% YoY. 매출 비중 12%. 과대평가되었다.

**"PEG 0.88이면 저평가 아닌가?"**
→ PEG는 함정이다. "영구적 성장"을 가정하지만, 현실은 FCF -$3B다.
