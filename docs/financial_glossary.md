# 재무 용어집

> 프로젝트에서 자주 사용하는 재무 지표 정리
> 단순 계산식이 아닌, **왜 중요한지**와 **어떻게 해석하는지**에 초점

---

## 밸류에이션 지표

### P/E (Price to Earnings Ratio) - 주가수익비율

```
P/E = 주가 ÷ 주당순이익(EPS)
    = 시가총액 ÷ 순이익
```

**의미**: 순이익 1원을 벌기 위해 얼마를 지불하는가

**왜 중요한가**:
- 가장 기본적인 밸류에이션 척도
- "이 회사가 현재 이익을 유지하면 몇 년 만에 투자금 회수?"라는 질문에 대한 답
- P/E 20 = 순이익이 유지되면 20년 후 투자금 회수

**해석 기준**:
| P/E | 일반적 해석 |
|-----|------------|
| < 10 | 저평가 또는 성장 정체 |
| 10-20 | 적정 |
| 20-30 | 성장 기대 반영 |
| > 30 | 고성장 기대 또는 과열 |

**주의**: P/E만 보면 안 되는 이유
- 적자 기업은 P/E 계산 불가
- 성장률을 반영하지 않음 → PEG로 보완
- 일회성 이익/손실에 왜곡됨

---

### P/S (Price to Sales Ratio) - 주가매출비율

```
P/S = 주가 ÷ 주당매출
    = 시가총액 ÷ 매출
```

**의미**: 매출 1원에 얼마를 지불하는가

**왜 중요한가**:
- 적자 기업도 평가 가능 (성장주, 테크 기업에 유용)
- 이익 조작이 어려운 매출 기준
- 같은 산업 내 비교에 적합

**해석 기준**:
| P/S | 해석 |
|-----|------|
| < 1 | 저평가 가능성 |
| 1-3 | 일반적 |
| 3-10 | 고성장 기대 |
| > 10 | SaaS/플랫폼 또는 과열 |

**주의**: 이익률이 낮으면 P/S가 낮아도 비쌀 수 있음

---

### P/B (Price to Book Ratio) - 주가순자산비율

```
P/B = 주가 ÷ 주당순자산(BPS)
    = 시가총액 ÷ 자기자본
```

**의미**: 장부상 순자산 1원에 얼마를 지불하는가

**왜 중요한가**:
- 회사를 청산하면 얼마 받을 수 있나? (최소 가치)
- 금융주, 자산 중심 기업 평가에 필수
- P/B < 1이면 "장부가보다 싸게 거래"

**해석 기준**:
| P/B | 해석 |
|-----|------|
| < 1 | 청산가치 이하 (딥밸류 또는 문제 있음) |
| 1-3 | 일반적 |
| > 3 | 무형자산/브랜드 가치 반영 |

**주의**:
- 테크 기업은 무형자산이 많아 P/B가 높음 (정상)
- 제조업은 유형자산이 많아 P/B로 비교 용이

---

### EV/EBITDA

```
EV (Enterprise Value) = 시가총액 + 순부채
                      = 시가총액 + 총부채 - 현금

EBITDA = 영업이익 + 감가상각비 + 무형자산상각비
       = 세전, 이자, 감가상각 전 이익

EV/EBITDA = EV ÷ EBITDA
```

**의미**: 회사 전체(부채 포함)를 인수한다면, 영업현금 창출력 대비 얼마를 지불하는가

**왜 중요한가**:
- **M&A의 표준 척도**: 회사를 통째로 살 때 쓰는 밸류에이션
- 자본구조(부채 비율)가 다른 기업 비교 가능
- 감가상각 정책 차이 제거 → 순수 영업 수익성 비교

**해석 기준**:
| EV/EBITDA | 해석 |
|-----------|------|
| < 6 | 저평가 |
| 6-10 | 적정 |
| 10-15 | 성장 기대 |
| > 15 | 고성장 또는 고평가 |

**P/E vs EV/EBITDA**:
- P/E: 주주 관점 (순이익)
- EV/EBITDA: 인수자 관점 (영업이익 + 부채 고려)

---

### PEG (Price/Earnings to Growth) - Lynch 핵심 지표

```
PEG = P/E ÷ EPS 성장률(%)

예: P/E 30, EPS 성장률 30% → PEG = 1.0
```

**의미**: 성장률 대비 P/E가 적정한가

**왜 중요한가**:
- P/E의 치명적 약점(성장률 무시) 보완
- **피터 린치의 핵심 도구**: "PEG < 1이면 저평가"
- 성장주 간 비교에 필수

**해석 기준** (Lynch):
| PEG | 해석 |
|-----|------|
| < 0.5 | 매우 저평가 |
| 0.5-1.0 | 저평가 (매수 고려) |
| 1.0 | 적정 |
| > 1.0 | 고평가 |
| > 2.0 | 매우 고평가 |

**주의**:
- 성장률은 과거? 미래 예상? 명확히 해야 함
- 성장률 0% 이하면 PEG 의미 없음
- 금리 환경에 따라 기준이 달라질 수 있음

---

### PEGY (PEG + Yield)

```
PEGY = P/E ÷ (EPS 성장률 + 배당수익률)

예: P/E 20, 성장률 10%, 배당 3% → PEGY = 20 ÷ 13 = 1.54
```

**의미**: 성장률 + 배당까지 고려한 밸류에이션

**왜 중요한가**:
- 배당주 평가 시 PEG보다 공정
- 저성장 고배당 vs 고성장 무배당 비교 가능

---

## 수익성 지표

### ROE (Return on Equity) - 자기자본이익률

```
ROE = 순이익 ÷ 자기자본 × 100
```

**의미**: 주주가 맡긴 돈으로 얼마나 벌었나

**왜 중요한가**:
- **버핏의 핵심 지표**: "ROE 15% 이상을 5년 이상 유지하는 기업"
- 주주 입장에서 가장 직관적인 수익성 척도
- 복리 효과의 원천: 높은 ROE = 빠른 내재가치 증가

**해석 기준** (Buffett):
| ROE | 해석 |
|-----|------|
| ≥ 20% | 탁월 |
| 15-20% | 우수 |
| 10-15% | 양호 |
| < 10% | 부진 |

**주의 - 고ROE의 함정**:
1. **부채로 뻥튀기**: 레버리지가 높으면 ROE도 높아짐
   - 부채/자본 비율과 함께 봐야 함
   - 부채 없이 높은 ROE가 진짜
2. **자사주 매입으로 분모 축소**: 자본이 줄면 ROE 상승
3. **일회성 이익**: 지속 가능한지 확인

**ROE = ROA × 레버리지**
- ROA가 낮은데 ROE가 높다? → 부채로 뻥튀기

---

### ROIC (Return on Invested Capital) - 투하자본이익률

```
ROIC = NOPAT ÷ Invested Capital
     = 세후영업이익 ÷ 투하자본

NOPAT = 영업이익 × (1 - 세율)
투하자본 = 순운전자본 + 순고정자산
        = 자기자본 + 유이자부채 - 현금
```

**의미**: 사업에 실제 투입된 자본이 얼마나 효율적으로 이익을 창출하는가

**왜 중요한가**:
- **멍거의 핵심 지표**: ROE보다 진실에 가까움
- 자본구조(부채 비율)에 덜 왜곡됨
- **ROIC > WACC**: 가치 창출의 핵심 공식
  - ROIC가 자본비용(WACC)보다 높아야 주주 가치 증가
  - ROIC < WACC면 사업할수록 손해

**해석 기준** (Munger):
| ROIC | 해석 |
|------|------|
| > 20% | 탁월한 자본 효율 |
| 15-20% | 우수 |
| 10-15% | 양호 |
| < 10% | 자본 파괴 가능성 |

**ROE vs ROIC**:
- ROE: 주주 자본 대비 (부채 효과 포함)
- ROIC: 전체 투하자본 대비 (순수 사업 효율성)
- 같은 사업이라면 ROIC가 더 정확한 비교

---

### 영업이익률 (Operating Margin)

```
영업이익률 = 영업이익 ÷ 매출 × 100
```

**의미**: 본업에서 매출 100원당 얼마 남기나

**왜 중요한가**:
- **Fisher의 핵심 관심사**: 이익률의 "추세"
- 가격 결정력과 운영 효율성의 증거
- 개선 추세 = 경쟁력 강화
- 악화 추세 = 경쟁 심화 또는 비용 통제 실패

**해석**: 절대 수치보다 **추세**와 **동종업계 비교**가 중요

---

### 매출총이익률 (Gross Margin)

```
매출총이익률 = (매출 - 매출원가) ÷ 매출 × 100
            = 매출총이익 ÷ 매출 × 100
```

**의미**: 제품/서비스 자체의 마진

**왜 중요한가**:
- **가격 결정력의 직접적 지표**
- 원가 대비 얼마나 비싸게 팔 수 있나
- 영업이익률보다 변동이 적어 구조적 경쟁력 파악에 유용

**산업별 차이**:
| 산업 | 일반적 매출총이익률 |
|------|-------------------|
| SaaS/소프트웨어 | 70-90% |
| 명품/브랜드 | 60-70% |
| 유통/소매 | 20-40% |
| 제조업 | 20-35% |

---

### 순이익률 (Net Margin)

```
순이익률 = 순이익 ÷ 매출 × 100
```

**의미**: 최종적으로 매출 100원당 얼마 남기나

**왜 중요한가**:
- 세금, 이자, 모든 비용 차감 후 진짜 수익성
- 하지만 이자 비용/세금 차이로 왜곡 가능

---

## 현금흐름 지표

### FCF (Free Cash Flow) - 잉여현금흐름

```
FCF = 영업현금흐름 - CapEx (자본적 지출)
    = 영업활동현금흐름 - 설비투자
```

**의미**: 사업 유지에 필요한 투자 후 남는 진짜 현금

**왜 중요한가**:
- **순이익의 함정 탈출**: 회계적 이익 vs 실제 현금
- 배당, 자사주 매입, 부채 상환, 인수에 쓸 수 있는 돈
- "Cash is King": 현금이 없으면 회계 이익은 허상

**FCF가 순이익보다 중요한 이유**:
1. 감가상각 정책으로 이익 조작 가능
2. 매출 인식 시점 조작 가능
3. 현금은 거짓말 안 함

**해석**:
- FCF 지속 양수: 건강한 사업
- FCF 지속 음수: 성장 투자 중 or 사업 모델 문제
- FCF > 순이익: 양질의 이익 (감가상각 多, CapEx 少)

---

### FCF Yield - 잉여현금흐름 수익률

```
FCF Yield = FCF ÷ 시가총액 × 100
```

**의미**: 시가총액 대비 얼마나 현금을 창출하나

**왜 중요한가**:
- 밸류에이션 + 현금창출력 동시 평가
- "이 회사를 사면 매년 몇 %의 현금을 돌려받나"
- 채권 수익률과 비교 가능

**해석 기준**:
| FCF Yield | 해석 |
|-----------|------|
| > 10% | 저평가 가능성 높음 |
| 5-10% | 양호 |
| 2-5% | 성장 기대 반영 |
| < 2% | 고평가 또는 성장주 |

---

### FCF 전환율 (FCF Conversion)

```
FCF 전환율 = FCF ÷ 순이익 × 100
```

**의미**: 순이익이 얼마나 현금으로 전환되나

**왜 중요한가**:
- **멍거의 품질 체크**: 100% 이상이면 양질의 이익
- 낮으면 → 매출채권 증가, 재고 증가, CapEx 과다 의심
- 높으면 → 감가상각 많고 유지 투자 적음 (좋은 사업)

**해석 기준** (Munger):
| FCF 전환율 | 해석 |
|------------|------|
| > 100% | 양질의 이익 (탁월) |
| 80-100% | 양호 |
| < 80% | 이익의 질 점검 필요 |

---

### Owner Earnings - 소유주 이익 (Buffett)

```
Owner Earnings = 순이익
               + 감가상각/상각
               - 유지 CapEx
               - 운전자본 증가분
```

**의미**: 주주가 실제로 가져갈 수 있는 돈

**왜 중요한가**:
- **버핏이 1986년 주주서한에서 제시**
- FCF보다 정교함: "유지 CapEx"만 차감
- 성장 CapEx는 제외 → 기존 사업의 진짜 수익력

**FCF vs Owner Earnings**:
- FCF: 모든 CapEx 차감
- Owner Earnings: 유지 CapEx만 차감 (성장 투자는 제외)
- 성장 기업은 Owner Earnings > FCF

**실무적 어려움**: 유지 CapEx vs 성장 CapEx 구분이 어려움

---

### CapEx (Capital Expenditure) - 자본적 지출

```
CapEx = 유형자산 취득 + 무형자산 취득
      = 설비, 건물, 토지, 소프트웨어 등 투자
```

**의미**: 미래를 위한 투자 지출

**유지 CapEx vs 성장 CapEx**:
- 유지 CapEx: 현재 사업 유지에 필요한 투자
- 성장 CapEx: 확장/신사업을 위한 투자

**왜 중요한가**:
- CapEx 많이 필요한 사업 = 자본 집약적 (capital intensive)
- 버핏이 선호하는 사업: 낮은 CapEx로 성장 가능 (자본 경량)

---

## 성장 지표

### EPS (Earnings Per Share) - 주당순이익

```
EPS = 순이익 ÷ 발행주식수
```

**의미**: 주식 1주당 벌어들이는 이익

**왜 중요한가**:
- 주식 수 변화를 반영한 이익
- 자사주 매입/유상증자 효과 반영
- P/E 계산의 분모

---

### EPS 성장률

```
YoY 성장률 = (올해 EPS - 작년 EPS) ÷ 작년 EPS × 100

CAGR (연평균성장률) = (최종값/초기값)^(1/연수) - 1
```

**의미**: 주당 이익이 얼마나 빨리 성장하나

**왜 중요한가**:
- **린치의 기업 분류 기준**
- 성장률에 따라 적정 P/E가 달라짐

| EPS 성장률 | 린치 분류 |
|-----------|----------|
| 20-50% | Fast Growers (공격적 성장주) |
| 10-20% | Stalwarts (대형 우량주) |
| < 10% | Slow Growers (배당주) |

---

### CAGR (Compound Annual Growth Rate) - 연평균 성장률

```
CAGR = (최종값/초기값)^(1/연수) - 1

예: 5년간 100 → 200
CAGR = (200/100)^(1/5) - 1 = 14.9%
```

**의미**: 복리로 환산한 평균 연간 성장률

**왜 중요한가**:
- 들쭉날쭉한 성장을 단일 수치로 표현
- 장기 성장 추세 파악
- 단순 평균보다 정확 (복리 효과 반영)

---

## 안정성 지표

### 부채비율 (Debt to Equity Ratio)

```
부채비율 = 총부채 ÷ 자기자본 × 100
```

**의미**: 자기 돈 대비 빚이 얼마나 많나

**왜 중요한가**:
- 재무 안정성의 기본 척도
- 높으면 → 이자 부담, 경기 침체 시 위험
- **버핏/린치 모두 저부채 선호**

**해석 기준** (Lynch):
| 부채비율 | 해석 |
|----------|------|
| < 30% | 매우 양호 |
| 30-50% | 양호 |
| 50-80% | 주의 |
| > 80% | 위험 |

**주의**: 산업별로 적정 수준 다름 (금융업은 높음이 정상)

---

### 순부채 (Net Debt)

```
순부채 = 총부채 - 현금 및 현금성 자산
```

**의미**: 현금으로 갚고 남는 실질 부채

**왜 중요한가**:
- 순부채 < 0 (순현금): 빚보다 현금이 많음 (안전)
- EV 계산에 사용: EV = 시총 + 순부채

---

### 이자보상배율 (Interest Coverage Ratio)

```
이자보상배율 = 영업이익 ÷ 이자비용
            = EBIT ÷ Interest Expense
```

**의미**: 영업이익으로 이자를 몇 번 갚을 수 있나

**왜 중요한가**:
- 부채 상환 능력의 직접적 지표
- 낮으면 → 이자 부담에 취약, 금리 인상 위험

**해석 기준**:
| 이자보상배율 | 해석 |
|-------------|------|
| > 10x | 매우 안전 |
| 5-10x | 양호 |
| 2-5x | 주의 |
| < 2x | 위험 |

---

### 부채/순이익 비율

```
부채/순이익 = 장기부채 ÷ 연간 순이익
```

**의미**: 현재 이익 수준으로 부채 상환에 몇 년 걸리나

**해석 기준** (Buffett):
- < 5년: 양호
- > 5년: 부채 부담 큼

---

## 기타 중요 지표

### 주식 희석률

```
희석률 = (현재 발행주식수 - 과거 발행주식수) ÷ 과거 발행주식수 × 100
```

**의미**: 주식 수가 늘어나 기존 주주 지분이 희석되는 정도

**왜 중요한가**:
- **멍거의 Red Flag**: 만성적 희석 = 주주 가치 파괴
- 스톡옵션, 유상증자로 발생
- EPS 성장 없이 주식 수만 늘면 실질 손해

**해석**:
- 감소 (자사주 매입): 긍정적
- 연 2% 이상 증가: 주의
- 연 5% 이상 증가: 심각한 희석

---

### 기관 보유율 (Institutional Ownership)

```
기관 보유율 = 기관투자자 보유 주식 ÷ 총 발행주식 × 100
```

**의미**: 전문 투자자들이 얼마나 보유하나

**왜 중요한가** (Lynch 관점):
- 낮으면 → 아직 덜 알려진 종목 (기회)
- 높으면 → 이미 발굴된 종목, 매도 압력 가능성

**린치 선호**: 기관 보유율이 **과도하지 않은** 종목

---

### LTV/CAC (단위 경제학)

```
LTV (Lifetime Value) = 고객 생애 가치
CAC (Customer Acquisition Cost) = 고객 획득 비용

LTV/CAC = 고객 1명의 평생 가치 ÷ 고객 획득 비용
```

**의미**: 고객 1명을 얻는 데 든 비용 대비 얼마나 벌어오나

**왜 중요한가**:
- **멍거의 단위 경제학**: 사업의 기본 수익 구조
- SaaS, 구독 모델, 플랫폼 비즈니스 평가에 필수

**해석 기준**:
| LTV/CAC | 해석 |
|---------|------|
| > 3x | 건강한 단위 경제학 |
| 1-3x | 개선 필요 |
| < 1x | 돈 버는 구조가 안 됨 |

---

### NOPAT (Net Operating Profit After Tax)

```
NOPAT = 영업이익 × (1 - 세율)
      = 세후 영업이익
```

**의미**: 이자 비용 제외, 세금만 반영한 영업이익

**왜 중요한가**:
- ROIC 계산에 사용
- 자본구조(부채)와 무관한 순수 영업 수익력

---

### 운전자본 (Working Capital)

```
운전자본 = 유동자산 - 유동부채
        = (현금 + 매출채권 + 재고) - (매입채무 + 단기부채)

순운전자본 변화 = 올해 운전자본 - 작년 운전자본
```

**의미**: 일상 영업에 묶여 있는 돈

**왜 중요한가**:
- 운전자본 증가 = 현금 유출 (성장 시 더 많은 돈이 묶임)
- 운전자본 감소 = 현금 유입 (효율적 관리)
- **Owner Earnings** 계산에 반영

**투자 판단 흐름**:

3. **성장성** 확인: 성장률이 지속 가능한가?
4. **현금흐름** 검증: 이익이 현금으로 전환되나?
5. **밸류에이션** 판단: 위 조건 충족 시 가격이 합리적인가?
